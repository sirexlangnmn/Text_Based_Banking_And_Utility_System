# Text-Based Banking & Utility System

A beginner-friendly Python project demonstrating core programming fundamentals through an interactive banking application.

## Features

- **Check Balance** - View your current account balance
- **Deposit** - Add funds to your account
- **Withdraw** - Remove funds (with insufficient funds validation)
- **Transaction Summary** - View all deposits and withdrawals
- **Interest Projection** - Calculate future balance with 2% monthly interest

## Prerequisites

- Python 3.x installed on your system

## How to Run

1. Open a terminal or command prompt
2. Navigate to the project directory
3. Run the script:
   ```bash
   python index.py
   ```
